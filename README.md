## 2. Key Bindings

### Mac

Prev line `Cmd+left`

Next line `Ctrl+right`

### Windows

Prev line `Cmd+left`

Next line `Ctrl+right`

### 激活、禁用快捷键（默认是禁用状态）

状态栏增加**Reading mode**（开启快捷键阅读功能）和**Coding mode**（禁用快捷键阅读功能）按钮,方便随时切换

在停止读书后会自动切换到`Coding`模式
